# Dictionary

The Dictionary component displays a list of key-value pairs.

<ComponentPreview
  componentName="dictionary--default-story"

  hasHTML
  titleAttr="Dictionary Example"
/>

### Usage

Use the Dictionary component when you have a small
set of data that you want to display using key-value
pairs. If you have a large set of data or more than
one value per key, consider using the Table component
instead.

### Guidelines

- There should be at least two key-value pairs.
- The length of the text in a key should not
exceed two lines of wrapped text.
- A Dictionary cannot be nested within another Dictionary.

## Anatomy

- Each row of a Dictionary contains a key on
the left and a value on the right.
- On small viewports, the value is displayed
below the key.
