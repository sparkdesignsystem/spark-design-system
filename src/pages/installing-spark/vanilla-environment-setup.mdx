## ğŸ›  Install Webpack

Now we are going to set up a tool called [Webpack](). Once we have
 Spark installed, we will use Webpack to combine all the JavaScript
 needed to run Spark (as well as any other JavaScript dependencies
  our website has) into one JavaScript file that we can include in
  our site.

Run this command to install Webpack.


## ğŸ¨ Install Sass

All the CSS for Spark is included in Sparkâ€™s `npm` package. In order

Sass is a tool that processes and compiles CSS. In our example,
weâ€™re going to use Sass to compile an `scss` file into a `css` file
that you can then reference in your site. In the next guide, weâ€™ll
 use Sass to import all the Spark CSS and include it in your site.

For more info on Sass, check out the [Sass Getting Started Guide]().

First, run this command to install all the packages you'll use to
 build the site CSS.

```
npm install node-sass sass-loader css-loader mini-css-extract-plugin --save-dev
```

In the root directory of your project, create a new file calledâ€¯
`webpack.config.js`â€¯and paste this code into it.

```
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

module.exports = {
  module: {
    rules: [
      {
        test: /\.s[ac]ss$/i,
        use: [
          MiniCssExtractPlugin.loader,
          'css-loader',
          'sass-loader',
        ],
      },
    ],
  },
  plugins: [
    new MiniCssExtractPlugin({
      filename: 'bundle.css'
    }),
  ],
};
```

This config file tells Webpack to do two things.

1. For any `scss` or `sass` files in the project, compile them first
using a tool called `sass-loader`, then with `css-loader`, and finally
with a tool called `MiniCssExtractPlugin`.

2. When `MiniCssExtractPlugin` finishes processing the CSS, put the
 output into a file called `bundle.css`.

At the top of `index.js`, add this reference to the uncompiled `scss`
 file.

```
import "../src/style.scss"
```

Now when Webpack compiles `index.js`, this import line will include
 your style file, which will be processed according to the rules
 we set up in the webpack config in the previous step.

Now letâ€™s write some CSS to compile.

Create a new file in `src` called `style.scss` and copy this CSS into it:

```

  background-color: green;
}
```

In `index.html`, add this reference to your new compiled CSS.

```

```
